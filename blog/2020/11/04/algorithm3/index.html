阶乘字符串

2020年11月4日 · 阅读需 5 分钟
DoremySwee
2022 届算法小组成员

这是 C 社算法团队的第三次活动。我们研究了一道上海省选的题目，主要运用了状态压缩动态规划算法。其中数学部分比较困难，尚未给出严格证明。

题目​

题目来源：洛谷 P3989

给定一个由前nnn个小写字母组成的串SSS。串SSS是阶乘字符串当且仅当前nnn个小写字母的全排列（共n!n!n!种）都作为的子序列（可以不连续）出现。

由这个定义出发，可以得到一个简单的枚举法去验证，但是它实在太慢了。所以现在请你设计一个算法，在 1 秒内判断出给定的串是否是阶乘字符串。

数据规模内存限制运行时间
∥S∥≤450\|S\|\le 450∥S∥≤450, n≤26n\le 26n≤26125MB1.0s

题解​

暴力解法自然是枚举全排列并检验，30%的分数到手，不过这个程序没有其它太大意义，甚至由于阶乘级的时间复杂度难以对拍。不多作讨论。

考虑 70%的分数。n≤20n\le 20n≤20，可以考虑O(2n)\mathcal{O}(2^n)O(2n)的算法。既然不能进行全排列的枚举，那么完全可以考虑状态压缩动态规划，枚举一个字母是否出现在状态中。由于要求解的是整个字符串中是否存在前nnn个字母的全排列，则考虑f(x)f(x)f(x)表示xxx对应的几个字母的全排列最早出现在字符串种的何处。状态转移方程举例：

f(a,b,c,e)=max⁡(从f(a,b,c)开始找第一个’e’,从f(a,b,e)开始找第一个’c’,从f(a,c,e)开始找第一个’b’,从f(b,c,e)开始找第一个’a’) \begin{aligned} f(a,b,c,e)=\max(&从f(a,b,c)开始找第一个\texttt{'e'}, \\ &从f(a,b,e)开始找第一个\texttt{'c'}, \\ &从f(a,c,e)开始找第一个\texttt{'b'}, \\ &从f(b,c,e)开始找第一个\texttt{'a'}) \end{aligned}f(a,b,c,e)=max(​从f(a,b,c)开始找第一个’e’,从f(a,b,e)开始找第一个’c’,从f(a,c,e)开始找第一个’b’,从f(b,c,e)开始找第一个’a’)​

但是如果对于每一个寻找都是通过循环寻找的话，时间会有点长，只能拿 50 分，需要进行一定的预处理。可以在预处理之后用二分查找，但是方便起见建议添加一个数组标记每一个位置下一个目标字母的位置，所需时间不长，450×26450\times 26450×26而已。

最后，考虑剩下的分数点。可以发现，对于过大的nnn，可以直接输出"No"，至于这个值如何确定？按照n=3  ⟹  S=abcbabn=3\implies S=\mathtt{abcbab}n=3⟹S=abcbab，n=4  ⟹  S=abcdcbabcdcban=4\implies S=\mathtt{abcdcbabcdcba}n=4⟹S=abcdcbabcdcba这种构造方式SSS的长度为n2−n+1n^2-n+1n2−n+1，n≤21n\le 21n≤21，不过证明过于困难。为保险起见，因为n=22n=22n=22时用时是可以接受的，取n≥23n\ge 23n≥23时直接输出"No"。

程序​

#include <iostream>

using namespace std;

int main(){
    int t;
    cin >> t;
    while (t--) {
        int n;
        cin >> n;
        string s;
        cin >> s;
        if (n >= 23) {
            cout << "NO" << endl;
        } else {
            short NEXT[26][450];
            for (int i = s.length() - 1; i >= 0; i--) {
                if (i == s.length() - 1) {
                    for (int j = 0; j < 26; j++)
                        NEXT[j][i] = -1;
                } else {
                    for (int j = 0; j < 26; j++)
                        NEXT[j][i] = NEXT[j][i + 1];
                }
                NEXT[s[i] - 'a'][i] = i;
            }
            short *f = new short[1 << n];
            for (int i = 0; i < (1 << n); i++) {
                f[i] = 0;
                for (int j = 0; (1 << j) <= i; j++) {
                    if (i & (1 << j)) {
                        if (NEXT[j][f[i - (1 << j)]] == -1) {
                            cout << "NO" << endl;
                            return 0;
                        }
                        f[i] = max(f[i], NEXT[j][f[i - (1 << j)]]);
                    }
                }
            }
            cout << "YES" << endl;
        }
    }
}
